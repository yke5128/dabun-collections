禁欲主義訣別宣言
2022/10/9

もう わたし がんばらない。
禁欲 と 忍従 を 是 と する 価値観 は 持って いません。
利己的 な 迎合 と 追従 を やめろ。
普通 の わたし を 誂える な。
社会 に 適合 する ため に 己 を 枉げる な。
わたし を 指弾 する 社会 が 蒙昧 な だけ だ。
ゆえに わたし は 普通 の 称号 を 得る ため に 努力 を しません。
努力 を 礼賛 したがる 社会 の 一方的 な 要求 と イデオロギー に 従い ません。
間違って いる のは わたし で なく 社会 で ある。
わたし を 間違って いる と 思う 連中 は 社会 に 洗脳 されて いる だけ で ある。
わたし は 社会 の 桎梏 から 絶対的 な 自由 を 勝ち取る ため に 闘って いる の である。
糾弾 できる もの なら して みろ。
わたし に 社会 一般 の 通念 を 説いても 糠 に 釘 で ある。
わたし を 矯正 できる もの なら して みろ。
死んでも 普通 には ならない ので。
新しい パースペクティヴ の 一つ でも 与えて みろ。
おまえら が 説ける 観念 ならば とっく に わたし は 知悉 して いる。
誰 にも わたし を 羈絆 させ や しない。
わたし を 縛り つけ たければ しろ。
そんな ちんけ な 鎖 直ちに 振り ほどいて 終わり なので。
わたし を 苦しめ たければ しろ。
巧妙 に 遁れる 策 を 弄する だけ なので。
社会 など わたし には 恐るる に 足らぬ。
陳腐 な 脅し文句 は わたし には 効かぬ。
おまえら の 術中 は 全て お見通し なのだ。
